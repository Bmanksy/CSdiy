#### const vs let vs var
var: 声明一个变量。可选初始化一个值；     
let: 声明一个块作用域的局部变量，可选初始化一个值；    
const: 声明一个块作用域的只读常量。

const cannot be changed
let can be changed at any point

let&const is block-scoped
var is function-scoped
